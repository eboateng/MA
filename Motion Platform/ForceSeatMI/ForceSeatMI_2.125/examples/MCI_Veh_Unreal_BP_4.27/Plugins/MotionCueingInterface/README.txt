DISCLAIMER:

This plug-in is implementation of "Motion Cueing Interface" that interfaces with ForceSeatMI and in the result with the motion platform.

Original "Motion Cueing Interface" has been created by Sebastien Loze and Francis Maheux - MotionSystems only implemented required calls of ForceSeatMI API.

If you have any questions how "Motion Cueing Interface" is supposed to work, please contact authors of the origial plugin:
https://github.com/ue4plugins/MotionCueingInterface

USAGE:
- copy ForceSeatMI directory into Plugins directory inside your project root directory
- copy MotionCueingInterface directory into Plugins directory inside your project root directory

For more details please take a look at provided examples.

MOTION CUEING INTERFACE - FROM GITHUB:

The Motion Cueing Interface plugin is a c++ code sample to help motion system manufacturers with the creation of a plugin that will interface with motion platforms. 

Developer and end user documentation resides in the Wiki.
